# Student Management System - QA Testing Project

## Overview

This project demonstrates manual testing of a Student Management System application.

The goal was to verify the functionality of different modules, identify defects, and document testing results using industry-standard QA practices.

## Modules Tested

- Login Module
- Add Student Module
- Search Student Module

## Testing Activities

- Functional Testing
- Validation Testing
- UI Testing
- Bug Reporting

## Deliverables

- Test Cases
- Test Execution Results
- Defect Reports
- QA Documentation

## Tools Used

- Microsoft Excel
- Manual Testing Techniques

## Skills Demonstrated

- Test Case Design
- Defect Identification
- Bug Reporting
- Software Testing Life Cycle (STLC)
- Quality Assurance Documentation
